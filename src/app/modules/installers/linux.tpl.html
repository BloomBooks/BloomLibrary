Getting Bloom On Linux

A beta release of Bloom 3.7 is currently available for Ubuntu Precise, Trusty, and Xenial. It can also be installed on Wasta12, Wasta14, and Wasta16. You are welcome to try other versions of Linux but we cannot promise to support them.

Step 1: Try to get the beta release of the Bloom Desktop program

sudo apt-get install bloom-desktop-beta

If this succeeds, you can skip to step 5 below (and you are living as a permanent beta tester of SIL software). If this fails because of authorization issues or a missing sudo program, you need to consult with the local Linux administrator. If this fails because apt-get cannot locate the package, then the following steps are needed.

Step 2: Add the SIL repository and the SIL experimental repository to your system

(for precise or wasta12):

wget -qO - http://packages.sil.org/sil.gpg | sudo apt-key add -
sudo add-apt-repository "deb http://packages.sil.org/ubuntu precise main"
sudo add-apt-repository "deb http://packages.sil.org/ubuntu precise-experimental main"
sudo apt-get update

(for trusty or wasta14):

wget -qO - http://packages.sil.org/sil.gpg | sudo apt-key add -
sudo add-apt-repository "deb http://packages.sil.org/ubuntu trusty main"
sudo add-apt-repository "deb http://packages.sil.org/ubuntu trusty-experimental main"
sudo apt-get update

(for xenial or wasta16):

wget -qO - http://packages.sil.org/sil.gpg | sudo apt-key add -
sudo add-apt-repository "deb http://packages.sil.org/ubuntu xenial main"
sudo add-apt-repository "deb http://packages.sil.org/ubuntu xenial-experimental main"
sudo apt-get update

Step 3: Get the beta release of the Bloom Desktop program

sudo apt-get install bloom-desktop-beta

Step 4: Remove the experimental repository from your system

(for precise)

sudo add-apt-repository --remove "deb http://packages.sil.org/ubuntu precise-experimental main"
sudo apt-get update

(for trusty)

sudo add-apt-repository --remove "deb http://packages.sil.org/ubuntu trusty-experimental main"
sudo apt-get update

(for xenial or wasta16)

sudo add-apt-repository --remove "deb http://packages.sil.org/ubuntu xenial-experimental main"
sudo apt-get update

(for wasta12 or wasta14)

synaptic

Using the graphical user interface, open the Settings / Repository dialog, then find and remove any entries in "Additional repositories" that include precise-experimental or trusty-experimental in the fine print.

Step 5: Copy the desktop launcher to your local area

cp -i /usr/share/applications/bloom.desktop ~/Desktop
chmod +x ~/Desktop/bloom.desktop

Step 6: Launch the Application

(mouse)

Double click the desktop launcher icon

(keyboard)

Type the name of the program in a command window or into the dashboard or startup menu

bloom
